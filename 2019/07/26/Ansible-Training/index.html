Ansible

Installation

  1. Install ansible in CentOS

    sudo yum -y install ansible

  2. Check Ansible version

    ansible --version

Basic command

  1. Create Inventory file, e.g. host

  2. Run with specify inventory
    ansible -i host host1 -m ping

  3. Modify the config file for one off change

  4. Ansible Playbook

  5. Run Ansible Playbook
    ansible-playbook test_playbook

  6. Run add-hoc command
    ansible webserver -m yum -a "name=httpd state=absent" --become
